# The 1-Bit Future

In February 2024, Microsoft Research published a paper on BitNet b1.58 — a language model where every weight is -1, 0, or 1. No floats. Just three values.

The claim: 1.58 bits per parameter achieves perplexity comparable to full-precision transformers at the same parameter count, while using 70% less energy and enabling integer-only matrix multiplication.

This breaks a decade of assumptions. The deep learning stack was built on the premise that gradients need float32 precision to converge. Quantization was always a post-hoc compression trick, trading quality for deployment cost.

BitNet suggests the opposite: extreme quantization might be *native* to how transformers learn, not a compromise. If true, the right model for local inference isn't "the biggest model that fits" — it's "the smallest quantization that still generalizes."

Your 1B model at Q4_K_M uses 4 bits per weight. BitNet suggests you could go further.
