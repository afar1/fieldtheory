# Contributing

Field Theory is preparing for a public source release under AGPL-3.0-or-later for Field Theory-owned app/source code.

By contributing to this repository, you agree that your contribution is licensed under the same license that covers the part of the repository you are changing.

For the app/source code, that means AGPL-3.0-or-later.

For examples or protocol documentation that is explicitly marked MIT, that means MIT.

**Project Boundaries**

This repository is for the public app/source surface.

Do not contribute production Supabase schema, migrations, edge functions, billing internals, deployment config, service secrets, private operational runbooks, or hosted Field Theory Cloud infrastructure.

Field Theory Cloud remains the official hosted service operated by Field Theory.

**Brand**

The code license does not grant rights to the Field Theory name, logo, icon, brand, or official Cloud/service identity.

Forks should use their own name and identity unless they have separate written permission.

See [TRADEMARKS.md](TRADEMARKS.md).

**Third-Party Code**

Do not add third-party code, assets, models, or generated media unless the source, license, and redistribution rights are documented.

Update [NOTICE.md](NOTICE.md) or [THIRD_PARTY_NOTICES.md](THIRD_PARTY_NOTICES.md) when a new notice is required.
