# Field Theory Tray Icon Assets

This directory contains macOS menu bar and packaged app icon assets for Field Theory.

## Current Files

- `fieldtheory-iconTemplate.png`
- `fieldtheory-iconTemplate@2x.png`
- `icon.icns`

Legacy `littleone-*` tray icons were removed during open-source readiness cleanup because the current tray code uses the Field Theory template icon for all tray states.

## Template Image Requirements

macOS template menu bar icons automatically adapt to the system light/dark mode and menu bar appearance.

Requirements:

- Use the `Template.png` suffix.
- Use black with alpha transparency only.
- Avoid colors and gradients.
- Standard size is 16x16 pixels.
- Retina size is 32x32 pixels as `*Template@2x.png`.
